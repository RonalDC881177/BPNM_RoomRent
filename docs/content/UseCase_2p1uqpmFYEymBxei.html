Aceptar visita
Steps
1. El Arrendador1 autenticado accede al sistema.
2. El Arrendador1 ingresa a la sección “Solicitudes de visitas”.
3. SYSTEM muestra la lista de solicitudes de visitas con la fecha y hora propuestas por los arrendatarios.
4. El Arrendador1 selecciona una solicitud pendiente.
5. SYSTEM muestra el detalle de la solicitud con las opciones “Aceptar”, “Rechazar” o “Proponer nueva fecha”.
6. El Arrendador1 selecciona la opción “Aceptar”.
7. SYSTEM actualiza el estado de la solicitud a “Aceptada”.
8. SYSTEM muestra un mensaje de confirmación: "La visita ha sido aceptada exitosamente".
9. El Arrendador1 recibe una notificación indicando que su visita fue aceptada.

2. Rechazar visita
Steps
1. El Arrendador1 autenticado accede al sistema.
2. El Arrendador1 ingresa a la sección “Solicitudes de visitas”.
3. SYSTEM muestra la lista de solicitudes de visitas con la fecha y hora propuestas por los arrendatarios.
4. El Arrendador1 selecciona una solicitud pendiente.
5. SYSTEM muestra el detalle de la solicitud con las opciones “Aceptar”, “Rechazar” o “Proponer nueva fecha”.
6. El Arrendador1 selecciona la opción “Rechazar”.
7. SYSTEM actualiza el estado de la solicitud a “Rechazada”.
8. SYSTEM muestra un mensaje de confirmación: "La visita ha sido rechazada".
9. El Arrendatario2 recibe una notificación indicando que su visita fue rechazada.

3 Proponer fecha
Steps
1. El Arrendador1 autenticado accede al sistema.
2. El Arrendador1 ingresa a la sección “Solicitudes de visitas”.
3. SYSTEM muestra la lista de solicitudes de visitas con la fecha y hora propuestas por los arrendatarios.
4. El Arrendador1 selecciona una solicitud pendiente.
5. SYSTEM muestra el detalle de la solicitud con las opciones “Aceptar”, “Rechazar” o “Proponer nueva fecha”.
6. El Arrendador1 selecciona la opción “Proponer nueva fecha”.
7. SYSTEM solicita que el arrendador ingrese una nueva fecha y hora disponibles.
8. El Arrendador1 ingresa la propuesta de fecha y confirma.
9. SYSTEM actualiza la solicitud con el nuevo horario y cambia el estado a “Fecha propuesta”.
10. SYSTEM muestra un mensaje de confirmación: "Se ha propuesto una nueva fecha de visita".
11. El Arrendatario2 recibe una notificación con la nueva fecha propuesta.
