2. Validacion de contraseñas
Steps
1. Cuando el Anonimo1 finaliza el registro de los datos iniciales de forma exitosa.
2. SYSTEM generara un nuevo apartado donde se solicitará contraseña y confirmación de contraseña.
3. Anonimo1 registra la información solicitada en los campos y da clic en el botón aceptar.
4. if Si la información registrada en los dos campos es igual y cumple con las características necesarias.
    4.1. SYSTEM permitirá avanzar a la confirmación del registro por medio del correo electrónico suministrado.
5. else
    5.1. SYSTEM Notificará que la información de los campos no es igual y debe corregirse para poder continuar.
   end if

3. Confirmación de registro.
Steps
1. Cuando el Anonimo1 completa el proceso de registro de contraseñas.
2. SYSTEM Creará la cuenta, enviará un correo de verificación a la dirección de correo registrada y generar un mensaje indicando “Tu cuenta fue creada exitosamente. Revisa tu correo para activarla”.
3. Anonimo1 da clic en el botón "Completar registro" .
4. if Anonimo1 recibe el correo para activar la cuenta
    4.1. Completa proceso de activación de la cuenta
5. else
    5.1. Anonimo1 da clic en el botón de "Reenviar correo de confirmación"
   end if

4. Activacion de cuenta
Steps
1. El Anonimo1 recibirá un correo para finalizar su registro con un botón para activar cuenta.
2. Anonimo1 da clic en el botón de activar cuenta
3. SYSTEM generara un modal con los términos y condiciones para ser aceptados por el cliente.
4. if El Anonimo1 le da clic en el botón aceptar.
    4.1. SYSTEM Finalizara el registro y el usuario ya podrá acceder a su cuenta
5. else
    5.1. SYSTEM No finalizara el registro y no se podrá acceder a la cuenta creada.
   end if
