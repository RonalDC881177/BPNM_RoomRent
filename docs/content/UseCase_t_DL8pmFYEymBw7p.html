1. Formulario de inicio de sesión
Steps
1. Usuario accede al sistema y selecciona la opción "Iniciar sesión".
Usuario accede al sistema y selecciona la opción "Iniciar sesión".
2. SYSTEM muestra el formulario de inicio de sesión con los campos: Correo electrónico y contraseña.
muestra el formulario de inicio de sesión con los campos: Correo electrónico y contraseña.
3. Usuario ingresa un correo electrónico, una contraseña da clic en el botón Ingresar.
Usuario ingresa un correo electrónico válido y una contraseña.
4. SYSTEM valida que ambos campos estén completos (no vacíos).
valida que ambos campos estén completos (no vacíos).
5. if Si uno o ambos campos están vacíos.
Si uno o ambos campos están vacíos.
    5.1. SYSTEM muestra un mensaje "Todos los campos son obligatorios".
muestra un mensaje "Todos los campos son obligatorios".
6. else if Si el correo electrónico o la contraseña no coinciden con los registros.
Si el correo electrónico o la contraseña no coinciden con los registros.
    6.1. SYSTEM muestra un mensaje de error "Correo o contraseña incorrectos".
muestra un mensaje de error "Correo o contraseña incorrectos".
7. else
    7.1. SYSTEM permite el acceso al usuario y muestra la página principal o el panel correspondiente.
permite el acceso al usuario y muestra la página principal o el panel correspondiente.
   end if

2. Gestión de seguridad
Steps
1. El Usuario ingresa de forma incorrecta los datos de inicio de sesión por mas de 5 veces.
El Usuario ingresa de forma incorrecta los datos de inicio de sesión por mas de 5 veces.
2. SYSTEM debe bloquear temporalmente la cuenta.
debe bloquear temporalmente la cuenta.
3. Si Usuario intenta intenta ingresar nuevamente.
Si Usuario intenta intenta ingresar nuevamente.
4. SYSTEM genera pop-up con el mensaje "Supero el numero máximo de intentos de ingreso cuenta se encuentra temporalmente bloqueada por 10 minutos luego podrá intentar acceder nuevamente o cambiar su contraseña".
genera pop-up con el mensaje "Supero el numero máximo de intentos de ingreso cuenta se encuentra temporalmente bloqueada por 10 minutos luego podrá intentar acceder nuevamente o cambiar su contraseña".
5. Cuando pase el tiempo de espera de la cuenta bloqueada.
6. SYSTEM Habilitara nuevamente el numero de intentos de ingreso.
Habilitara nuevamente el numero de intentos de ingreso.
